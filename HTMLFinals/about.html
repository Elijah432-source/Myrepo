About Shopping Mart

This project is a demonstration of integrating a public API to showcase dynamic product data.

API Information

The data displayed on this website is fetched from the Fake Store Api. It provides a simple JSON-based API for products, including details such as:

  • Product titles
  • Prices
  • Images
  • Categories

Features

  • Dynamic product listing with search and category filtering.
  • Modern JavaScript functionality for fetching and displaying data.
